# SimpleBotScenario

Реализация телеграмм бота.

 - Функции
    - Возможность отправлять сообщения пользователю.
    - Возможность регистрировать и сохранять сообщения от пользователя.
    - Возможность регистрировать и сохранять сообщения от каналов.
- Технологии
    - EF Core
    - PostgreSQL
    - C#
